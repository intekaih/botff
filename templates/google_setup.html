Cài đặt Google OAuth

Để dùng tính năng Login bằng Google, bạn cần tạo OAuth credentials từ Google Cloud Console (miễn phí, ~5 phút).

Bước 1 — Tạo Google OAuth Credentials
1. Vào console.cloud.google.com/apis/credentials
2. Tạo project mới (hoặc chọn project có sẵn)
3. Click + CREATE CREDENTIALS → OAuth client ID
4. Application type: chọn Web application
5. Phần Authorized redirect URIs → Add tất cả các URI sau:
{% for uri in all_uris %} {{ uri }} {% endfor %} (mỗi URI dán vào 1 dòng riêng)
6. Click CREATE → Copy Client ID và Client Secret
Bước 2 — Nhập credentials vào đây
✅ Đã lưu!
💡 Lưu ý: Nếu Google hiển thị cảnh báo "App chưa được xác minh" → click Advanced → Go to... (unsafe) → không sao cả, đây là app cá nhân của bạn.
