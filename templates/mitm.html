Cách hoạt động

Script auto_extract_mitm.py đóng vai trò như một proxy ngồi giữa điện thoại và server Garena. Khi game gửi request đăng nhập, proxy chặn response và tự động trích xuất JWT Token.

📱 Điện thoại
  │ (Wi-Fi → đặt SOCKS/HTTP proxy)
  ▼
🖥️ mitmproxy (chạy trên server này)
  │ ← chặn response /MajorLogin
  │ ← trích xuất JWT Token
  │ ← lưu vào access.txt
  ▼
🌐 Server Garena / Free Fire
⚠️ Bắt buộc: Điện thoại phải kết nối proxy TRƯỚC KHI mở Free Fire và đăng nhập. Nếu đăng nhập xong rồi mới bật proxy → sẽ không capture được.
ℹ️ Script này intercept endpoint /MajorLogin của Garena — đây chính là nơi server trả về JWT Token sau khi xác thực Google/Facebook/Guest thành công.
📋

Yêu cầu

✅ mitmproxy & blackboxprotobuf (đã cài)
✅ Điện thoại và server cùng mạng hoặc server có IP public
✅ Cài CA Certificate của mitmproxy lên điện thoại
✅ Free Fire chưa chạy / chưa đăng nhập khi bật proxy
⚠️ Android 7+ cần root hoặc dùng apk patch để bypass SSL Pinning của FF
📖

Hướng dẫn từng bước

1
Khởi động mitmproxy trên server
Chạy lệnh sau trong terminal:
cd tools/mitm_scripts mitmdump -s auto_extract_mitm.py --listen-port 8080
mitmproxy sẽ lắng nghe ở port 8080
2
Cài đặt proxy trên điện thoại
Vào Cài đặt Wi-Fi → Sửa mạng → Proxy thủ công:
Loại:  HTTP
Host:  [IP server hoặc IP máy tính]
Port:  8080
3
Cài CA Certificate (QUAN TRỌNG)
Mở trình duyệt trên điện thoại → truy cập:
http://mitm.it
Tải và cài certificate → Cài đặt → Bảo mật → Chứng chỉ
4
Mở Free Fire → Đăng nhập
Chọn đăng nhập bằng Google / Facebook / Garena.
Proxy phải bật TRƯỚC khi đăng nhập!
5
Token được tự động lưu
Terminal sẽ hiển thị token và lưu vào:
tools/bot/data/access.txt
✅ Sau khi lưu xong → dùng bình thường cho Buff Like

dump.py — Mod Skin Realtime

Script dump.py can thiệp vào response game để inject skin, emote vào client. Chỉ thấy ở phía bạn, không ảnh hưởng server hay người khác.

Tính năng Endpoint bị chặn
💎 VIP Bypass GetPrimeAccountInfo
🎒 Backpack đầy đồ GetBackpack
💃 Emote tất cả ChooseEmote
🔫 Skin súng trong trận ChooseSlotsAndShow
👗 Skin nhân vật trong trận GetPlayerPersonalShow
⚠️ Cũng cần kết nối proxy + cài CA cert trước. Giống hệt bước 1-3 của tab "Lấy Token". Sau khi kết nối proxy → mở game → vào loby → skin sẽ hiển thị.
📖

Hướng dẫn chạy Mod

1
Thêm ID skin vào tab "Quản lý ID"
ID skin quần áo (bất kỳ số), ID skin súng phải bắt đầu bằng 907
2
Chạy dump.py qua mitmdump
cd tools/mitm_scripts mitmdump -s dump.py --listen-port 8080
3
Kết nối điện thoại vào proxy
Cùng quy trình như tab Lấy Token (bước 2-3). CA cert chỉ cần cài 1 lần.
4
Mở game → Skin xuất hiện tự động
Vào lobby → kho đồ → mặc đồ từ backpack. Trong trận skin sẽ render.
