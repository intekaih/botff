🌐 Lấy Token Acc Thật qua Web (không cần điện thoại)

iOS / Android
📋 Hướng dẫn lấy Session Key + Open ID
1. Mở trình duyệt máy tính → vào https://sso.garena.com
2. Đăng nhập bằng Google / Facebook / Garena
3. Nhấn F12 → tab Application → Cookies → https://sso.garena.com
4. Tìm cookie session_key → copy giá trị
5. Tìm cookie open_id → copy giá trị
6. Dán vào ô bên dưới → bấm Lấy Token
⚠️ Nếu sso.garena.com không có cookie cần tìm → thử đăng nhập qua https://account.garena.com rồi xem cookie ở domain .garena.com
📡 Lấy Access Token từ Network Tab
1. Vào https://sso.garena.com → đăng nhập
2. F12 → tab Network → lọc connect.garena.com
3. Tìm request token/grant → xem phần Response
4. Copy giá trị access_token và open_id
