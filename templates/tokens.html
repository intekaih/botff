🤖 Lấy Token qua ADB (Android Debug Bridge)

Android Root Không cần MITM
ADB là gì? Android Debug Bridge — công cụ của Google để kết nối máy tính với điện thoại Android qua USB/WiFi.
Với điện thoại đã root, ADB cho phép đọc thẳng thư mục ứng dụng Free Fire → lấy token lưu trong bộ nhớ mà không cần chặn mạng.
✅ Hoạt động với mọi loại tài khoản (Google, Facebook, Garena)
✅ Không cần certificate MITM
⚠️ Yêu cầu điện thoại đã root + bật USB Debugging
📥 Bước 0 — Cài ADB trên máy tính
🪟 Windows
winget install Google.PlatformTools
hoặc tải platform-tools từ Google
🍎 macOS
brew install android-platform-tools
🐧 Linux
sudo apt install adb
📱 Bước 1 — Bật USB Debugging trên điện thoại
Cài đặt → Thông tin điện thoại → tap "Số build" 7 lần → Tuỳ chọn nhà phát triển (Developer Options) → bật USB Debugging
Cắm USB vào máy tính → Điện thoại hiện hộp thoại → chọn "Luôn cho phép"
💻 Bước 2 — Kết nối & kiểm tra
adb devices # Kết quả đúng: dòng "xxxxxxxxxx device" # Nếu "unauthorized" → xác nhận trên điện thoại
🔍 Bước 3 — Extract token (kiểm tra FF thường + FF Max)
⚠️ Windows CMD: Multi-line script không hoạt động với MagiskSU. Dùng Cách A (push file) bên dưới — hoạt động trên mọi OS.
Tạo file ff_extract.sh trên máy tính với nội dung sau, rồi push lên điện thoại và chạy:
📋 Nội dung file ff_extract.sh:
⬇ Tải file về luôn
#!/system/bin/sh for PKG in com.dts.freefireth com.dts.freefiremax; do DIR=/data/data/$PKG if [ -d "$DIR" ]; then echo "" echo "==== $PKG ====" find $DIR/shared_prefs -name '*.xml' 2>/dev/null | while read f; do if grep -qi 'access_token\|open_id\|garena_token' "$f" 2>/dev/null; then echo "[FILE] $f" grep -i 'access_token\|open_id\|garena_token' "$f" 2>/dev/null fi done else echo "$PKG: not installed" fi done
▶️ Sau khi lưu file, chạy 3 lệnh này trong CMD / Terminal:
# 1. Push script lên điện thoại adb push ff_extract.sh /sdcard/ff_extract.sh # 2. Chạy với quyền root adb shell su -c "sh /sdcard/ff_extract.sh" # 3. (Tuỳ chọn) Kéo toàn bộ SharedPrefs về máy để xem file adb shell su -c "cp -r /data/data/com.dts.freefireth/shared_prefs /sdcard/ff_prefs 2>/dev/null; cp -r /data/data/com.dts.freefiremax/shared_prefs /sdcard/ffmax_prefs 2>/dev/null" adb pull /sdcard/ff_prefs ./ff_dump adb pull /sdcard/ffmax_prefs ./ffmax_dump
Mở game trước, rồi chạy lệnh logcat này — bắt token ngay khi game gửi lên server:
# Windows CMD: adb logcat -v time | findstr /i "access_token open_id token_grant" # Linux / macOS / PowerShell: adb logcat -v time | grep -i "access_token\|open_id\|token_grant" # Lưu log ra file (Windows): adb logcat -v time > ff_log.txt # Sau đó mở ff_log.txt và tìm access_token
✅ Bước 4 — Dán token lấy được → nhận JWT game
Tìm trong output ADB:
• Dòng có access_token → copy value
• Dòng có open_id → copy value (số)
