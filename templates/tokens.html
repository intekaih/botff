🤖 Lấy Token qua ADB (Android Debug Bridge)

Android Root Không cần MITM
ADB là gì? Android Debug Bridge — công cụ của Google để kết nối máy tính với điện thoại Android qua USB/WiFi.
Với điện thoại đã root, ADB cho phép đọc thẳng thư mục ứng dụng Free Fire → lấy token lưu trong bộ nhớ mà không cần chặn mạng.
✅ Hoạt động với mọi loại tài khoản (Google, Facebook, Garena)
✅ Không cần certificate MITM
⚠️ Yêu cầu điện thoại đã root + bật USB Debugging
Chạy từng lệnh dưới đây trên máy tính (sau khi cài ADB):
📥 Bước 0: Cài ADB trên máy tính
Windows
winget install Google.PlatformTools
hoặc tải platform-tools từ Google
macOS
brew install android-platform-tools
Linux (Ubuntu)
sudo apt install adb
📱 Bước 1: Bật USB Debugging trên điện thoại
Cài đặt → Thông tin điện thoại → tap "Số build" 7 lần → Tuỳ chọn nhà phát triển → bật USB Debugging
💻 Bước 2: Kết nối & kiểm tra
# Cắm USB vào máy tính → chạy lệnh này: adb devices # Kết quả mong đợi: "List of devices attached" # và dòng "xxxxxxxxxx device" # Nếu hiện "unauthorized" → xác nhận trên điện thoại
🔍 Bước 3: Tìm token trong Free Fire (cần root)
# === CÁCH 1: Script tự động (copy toàn bộ, paste 1 lần) === adb shell su -c " FF_DIR=/data/data/com.dts.freefireth echo '=== SharedPreferences ===' find \$FF_DIR/shared_prefs -name '*.xml' -exec grep -l 'access_token\|open_id\|session_key\|token' {} \; 2>/dev/null echo '=== Nội dung files liên quan ===' find \$FF_DIR/shared_prefs -name '*.xml' -exec grep -i 'access_token\|open_id\|session_key\|garena_token' {} \; 2>/dev/null echo '=== Files JSON ===' find \$FF_DIR/files -name '*.json' 2>/dev/null | head -10 " # === CÁCH 2: Pull toàn bộ shared_prefs về máy (xem offline) === adb shell su -c "cp -r /data/data/com.dts.freefireth/shared_prefs /sdcard/ff_prefs" adb pull /sdcard/ff_prefs ./ff_prefs_dump # Sau đó mở thư mục ff_prefs_dump và tìm file chứa access_token # === CÁCH 3: Dùng logcat bắt khi game đang chạy === adb logcat -s GarenaSDK:* | grep -i "access_token\|open_id\|token_grant"
📋 Kết quả cần tìm:
access_token — chuỗi dài ~200-400 ký tự
open_id — dãy số (Garena UID của bạn)
Sau khi có 2 giá trị này → nhập vào ô

⚠️ Không root: Free Fire chặn ADB backup (FLAG_ALLOW_BACKUP = false).
Cách dưới đây chỉ hoạt động nếu điện thoại có Android 11 trở xuống và game chưa được cập nhật phòng chống backup.
# Thử backup (có thể bị block): adb backup -noapk com.dts.freefireth -f ff_backup.ab # Nếu tạo được file, convert sang tar: dd if=ff_backup.ab bs=24 skip=1 | python3 -c "import zlib,sys; sys.stdout.buffer.write(zlib.decompress(sys.stdin.buffer.read()))" > ff_backup.tar tar xf ff_backup.tar # Tìm file shared_prefs/*.xml # === Phương án không cần root: dùng account.garena.com === # Mở https://account.garena.com → Login → F12 → Network # Tìm request đến 100067.connect.garena.com # Lấy access_token + open_id từ response JSON
Kết nối ADB qua WiFi (không cần cắm USB sau lần đầu):
# === Lần đầu: cắm USB để setup === adb tcpip 5555 adb connect [IP_ĐIỆN_THOẠI]:5555 # Xem IP điện thoại: Cài đặt → WiFi → [tên mạng] → chi tiết # === Sau đó rút USB, dùng qua WiFi === adb connect [IP_ĐIỆN_THOẠI]:5555 adb devices # Chạy các lệnh extract như tab "Có Root" # === Android 11+: Wireless Debugging (không cần USB lần đầu) === # Cài đặt → Tuỳ chọn nhà phát triển → Wireless debugging # Pair bằng mã PIN → adb pair [IP]:[PORT]
